## 🏷 Commit Convention

**형식**: `날짜-[주제]-내용-상태`

```
example: 2025-09-05 [Network] TCP 3-way handshake 정리 Add
```

* 날짜 : YYYY-MM-DD
* 주제 : \[OS], \[DB], \[Algo], \[Network], \[Docs], \[Web] …
* 상태 : Add / Update / Fix / Delete
